# NovaScript 官方参考文档

NovaScript 是一种专为高级像素画生成、图层控制与逐帧动画设计的领域特定语言 (DSL)。
它全面拥抱了 Pythonic (类 Python) 风格，去除了繁琐的引号和花括号，引入了严格的缩进控制流与原生颜色类型，为您提供极其优雅的创意编程体验。

---

## 1. 核心设计哲学

* 极简至上：没有分号，没有花括号。用换行表示语句结束，用缩进（空格或 Tab）表示代码块。
* 原生色彩：十六进制颜色（HEX）是一等公民，无需加引号包裹。
* 数学驱动：坐标和尺寸完美支持基础代数运算。
* 双向同步：您手写的代码会生成像素画；您在画布上手绘的图案，也会通过底层的贪婪网格算法实时转化为极其紧凑的代码。

---

## 2. 基础语法与数据结构

### 2.1 数据类型
NovaScript 包含四种基础数据类型？： 

**1. 数字 (Number)**：仅支持整数，如 10, 20。NovaScript 是纯粹的像素画语言，没有模糊的浮点数，所有的坐标、计算全部都是纯整数。除法计算会自动向下取整，例如 9 / 2 结果为 4。 

**2. 颜色 (Color)**：原生十六进制颜色。支持 6 位纯色或 8 位带透明度颜色，必须以 # 开头。例如：#FF0000 (红)，#80000000 (半透明黑)。 

**3. 字符串 (String)**：用于为内存快照（图章）命名，使用双引号或单引号包裹。支持使用 + 运算符进行动态拼接（例如："frame_" + 1 结果为 "frame_1"）。 

**4. 列表 (List)**：使用中括号[] 包裹的一组数据，支持索引访问。例如：sizes =[10, 20, 30]，通过 sizes[0] 可获取 10。

### 2.2 变量声明
直接使用 = 赋值，无需关键字，变量名除了不可重复外无任何限制：
```python
SKY_COLOR = #3399FF
radius = 15
palettes =[#FF0000, #00FF00, #0000FF]
```

### 2.3 系统内置常量
引擎在每次执行时，会自动注入当前画布的尺寸常量，它们是严格只读的，并且大小写脱敏：
* W 或 w：当前画布的绝对宽度。
* H 或 h：当前画布的绝对高度。

尺寸设置原则：如果想让图形铺满全屏，请使用 w 和 h；如果想绘制固定大小的方块，请直接填入数值。切勿尝试给系统常量重新赋值。

### 2.4 注释
由于 # 号被赋予了原生颜色的含义，请使用双斜杠 // 进行单行注释。

## 3. 数学与表达式

所有的坐标、尺寸和循环次数等参数，都支持动态的数学表达式。
* 基础运算符：+ (加, 及字符串拼接), - (减), * (乘), / (除)。
* 取模运算：% (取模)。这对生成周期性图案极其有用。
* 比较运算符：==, !=, <, >, <=, >= (用于条件判断)。
* 单目减法：支持直接使用负数，例如 -5。
* 内置核心函数：
  * len(list)：获取列表的长度。
  * abs(num)：获取绝对值。
  * max(a, b)：获取两者的最大值。
  * min(a, b)：获取两者的最小值。
  * clamp(val, min, max)：将数值钳制在指定的 [min, max] 区间内，在处理像素边界时极其有用。

## 4. 控制流与复用

### 4.1 repeat 循环与 as 别名
在 repeat 块内，系统会自动注入一个名为 i 的递增变量（从 0 开始）。
如果需要多层嵌套循环，强烈建议使用 as 关键字为迭代器命名，以防止变量名冲突。内层循环结束后，外层的同名变量会自动物理恢复，绝不污染。
```python
repeat 8 as x:
    repeat 8 as y:
        dot(x * 10, y * 10, #FFFFFF)
```

### 4.2 if / else 条件分支
支持标准的条件判断，冒号后需缩进。
```python
repeat 10:
    if i % 2 == 0:
        dot(i * 5, 10, #FFFFFF)
    else:
        dot(i * 5, 10, #FF0000)
```

### 4.3 macro 宏定义 (动作宏与计算宏)
支持极简的自定义封装，支持传参且作用域完全隔离。宏分为两种：

1. 动作宏（以 : 结尾）：封装一系列绘制动作，无返回值。
```python
macro drawEye(ex, ey):
    offset ex, ey:
        circle(0, 0, 3, #FFFFFF)
        dot(0, 0, #000000)

drawEye(10, 15)
```

2. 计算宏（以 -> 结尾）：不包含代码块，直接返回一个计算结果。
```python
macro cx() -> w / 2
macro cy() -> h / 2
circle(cx(), cy(), 10, #FF0000)
```

## 5. 核心绘图 API

绝大多数绘图指令遵循 (x, y, [尺寸参数], color) 的格式。

* clear()：清空当前帧的整个画布，将其变为全透明。
* dot(x, y, color)：在 (x, y) 坐标处绘制一个 1x1 的像素点。
* line(x1, y1, x2, y2, color)：使用 Bresenham 算法绘制一条完美的无锯齿像素线段。
* rect(x, y, w, h, color)：以 (x, y) 为左上角起点，绘制一个宽 w、高 h 的填充矩形。(同义词：fillRegion)
* circle(x, y, r, color)：以 (x, y) 为中心点，以 r 为半径，绘制一个实心圆。(同义词：drawCircle)
* erase(x, y, w, h)：擦除一个区域，将其挖空恢复为全透明状态。

## 6. 空间变换与镜像

* offset dx, dy: ：安全局部坐标系上下文（强烈推荐）。在缩进块内，原点会偏移 (dx, dy)。离开缩进块时，原点会被系统强制物理还原，绝不污染后续代码！
* shiftDraw(dx, dy)：全局修改坐标系原点（保留向下兼容）。会产生全局污染，恢复正常请务必调用 shiftDraw(0, 0)。
* flipX() / flipY()：将当前帧画布上已有的所有像素进行全局水平/垂直镜像翻转。
* movePixels(dx, dy)：将当前帧画布上已有的所有像素整体平移，超出画布部分将被裁剪。
* mirrorX axis: / mirrorY axis: ：局部镜像上下文。在此代码块内的所有绘制行为，都会自动以 axis 指定的坐标为轴心生成对称镜像，支持嵌套使用。
```python
mirrorX 15:
    rect(10, 5, 2, 2, #FFFFFF) 
    // 底层会自动在 x=18 的对称位置同时绘制
```

## 7. 内存、图层与图章

这一组 API 是制作复杂逐帧动画的神器，允许在内存中暂存浮动图层，并随时调用。得益于动态字符串拼接，现在可以通过循环实现极其复杂的生成模式。

* save("name")：将当前画布的所有画面存在内存快照中。
* load("name")：清空当前画布，并将对应的快照完美覆盖到画布上。
* stamp("name", x, y)：将快照像图章一样叠加印在当前画布的 (x, y) 位置。

```python
// 动态命名示例
repeat 5 as frameId:
    save("cache_layer_" + frameId)
```

---

## 8. 动画与元数据设定

* copyFrame()：结束对当前帧的绘制，进入下一帧，并将当前帧的所有画面完全复制到下一帧中。
* newFrame()：结束对当前帧的绘制，进入下一帧，下一帧是一张全新的全透明空白画布。
* setFps(value)：设定动画的基础帧率 (1~60)。
* setHolds(value)：设定单帧驻留拍数 (1~10)。例如设为 3 代表一帧画面会在屏幕上停留 3 个计算周期。

## 9. 综合实战范例

以下代码展示了如何利用计算宏、offset 局部上下文、动态字符串与图章生成一个逐帧缩放的动画：

```python
setFps(20)
setHolds(3)

// 1. 定义计算宏，获取屏幕中心
macro cx() -> w / 2
macro cy() -> h / 2

// 2. 利用循环生成 3 帧基础元素，存入带有动态序号的内存中
repeat 3 as idx:
    clear()
    rect(0, 0, w, h, #1A1A24)
    circle(cx(), cy(), 10 + idx * 5, #3399FF)
    save("bg_layer_" + idx)

clear()

// 3. 安全局部坐标系与完美钳制
offset cx() - 10, cy() - 10:
    repeat w as px:
        repeat h as py:
            if clamp(px, 0, 20) == px:
                if clamp(py, 0, 20) == py:
                    dot(px, py, #FFFFCC)

save("star_layer")

// 4. 组合成动画帧
repeat 3 as f:
    if f > 0:
        newFrame()
    load("bg_layer_" + f)
    stamp("star_layer", 0, 0)
```
